HOW TO RUN:
-------------------------------------------------------
Method 1 (Recommended):
Double click on "RUN_PROJECT.bat". It will automatically check requirements and launch the app.

Method 2 (Manual):
Open the 'Bin' folder and run 'project.exe'.
-------------------------------------------------------
REQUIREMENTS:
-------------------------------------------------------
1. OS: Windows 10 or Windows 11.
2. Internet Connection: Required for AI Mode (Gemini API).
3. Output: The AI mode generates an HTML file using a browser for better formatting.
-------------------------------------------------------
COMPILATION (If you want to re-compile):
-------------------------------------------------------
If you wish to compile the source code in 'Source Code/main.c':
Command: gcc main.c -o ../Bin/project.exe -lcomdlg32

Note: -lcomdlg32 is required for the File Selection Dialog box.
